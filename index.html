Project Title: Implementation of the Parallel Algorithm for Planar Subgraph Isomorphisms

Name: Marc Vicuna

E-Mail: vicuna.marc@gmail.com

Project Outline: My research consists of the implementation of the Subgraph Isomorphism detection algorithm explained in Parallel Planar Subgraph Isomorphism and Vertex Connectivity . Moreover, I will confirm empirically whether the described bounds calculated in the paper ressemble the ones observed in practice, searching for both worst case performance as well as average performance. This implementation is not meant for cutting-edge industry-level performance, rather, it aims to test the practicality and efficiency of the algorithm for practical uses in an open-source way.
